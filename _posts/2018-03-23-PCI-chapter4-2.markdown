# 搜索引擎
一个简单的搜索引擎主要有以下三个部分组成：

    网页抓取：从一个或一组特定的网页开始，根据网页内部链接逐步追踪到其他网页。
             这样递归进行爬取，直到到达一定深度或达到一定数量为止。

    建立索引：建立数据表，包含文档中所有单词的位置信息，文档本身不一定要保存到数据库中，
             索引信息只需简单的保存到一个指向文档所在位置的引用即可。

    查询和排名：根据合适的网页排序方法，返回一个经过排序的页面列表。
